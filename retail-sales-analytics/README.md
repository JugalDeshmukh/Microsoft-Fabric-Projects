# Retail Sales Analytics Platform – Microsoft Fabric

## Overview
End-to-end analytics project built using Microsoft Fabric to transform raw retail data into business insights.

## Architecture
Dataflow Gen2 → Fabric Warehouse → Semantic Model → Power BI

## Key Features
- Data ingestion from parquet files
- Data cleansing and deduplication using Dataflows Gen2
- Star schema modeling (fact_orders, dim_users, dim_products)
- Interactive Power BI dashboards for revenue and customer analysis

## Tech Stack
- Microsoft Fabric
- Dataflows Gen2
- Fabric Warehouse
- Power BI
- Power Query
- SQL
